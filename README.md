#Chatbot Donas CETI 🍩

Grupo 6E — Inteligencia Artificial
Proyecto de chatbot desarrollado para la materia de Inteligencia Artificial.
El sistema simula la atención al cliente de una tienda ficticia de donas llamada Donas CETI. El proyecto demuestra cómo aplicar IA y Deep Learning para crear un chatbot funcional, capaz de clasificar intenciones y responder preguntas de atención al cliente.

Combina dos enfoques:

Modelo de Deep Learning (TensorFlow/Keras) para clasificar intenciones.
Chatbot conversacional que responde según la intención detectada.
El entrenamiento del modelo se realizó en Google Colab debido a compatibilidad de TensorFlow.

🧠 Funcionalidad del Chatbot El usuario escribe texto libre y el sistema identifica una intención, como:

saludo
menú
precios
horario
ubicación
despedida
Luego el chatbot responde adecuadamente según el caso.
